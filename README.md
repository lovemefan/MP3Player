## 简单的MP3播放器
__`作者:lovemefan`__
__`上传时间:2017/11/16`__
**实现的功能**:
* **1.**暂停,开始音乐,后台播放
* **2.**进度条可拖动,能实现音乐跳转
* **3.**能够循环切换上一曲和下一曲
* **4.**播放完毕后自动下一曲
* **5.**歌曲封面有旋转的动画
* **6.**背景图片毛玻璃美化

**待实现的功能**
* 本地音乐读取播放
* 网络音乐播放
 
**工程说明:**
> com.lovemefan.mp3player
>>BlurImage.java-----------------------实现毛玻璃效果
CircleImageView.java -------------圆形ImageView控件
MainActivity.java---------------------主Activity
Music.java-----------------------------音乐信息类
MusicPlayService.java--------------音乐服务类
